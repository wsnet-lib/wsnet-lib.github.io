Installation

Install the client (GameMaker Studio)#

  1. Download the extension from https://github.com/wsnet-lib/gms-client and unzip it.

  2. Run Game Maker Studio and import the file gws_2_3.yymps (by dragging the file on the editor and importing all resources)

  1. There you go, the game will now be able to connect to your multiplayer server on your machine. If you want to test the connection, please proceed with the server installation.

Install the server#

  1. Download the server from https://github.com/wsnet-lib/server and unzip it.

  2. Install Node.js from https://nodejs.org/en

  3. In the server folder, open a terminal window (eg. PowerShell) and run:

$ npm install
$ npm run watch
